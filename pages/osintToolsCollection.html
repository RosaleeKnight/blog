OSINT Tools Collection | Oct 28, 2023

While shifting through LinkedIn Posts I came across an interesting, curated tool list in the comments of LetsDefend’s Cybersecurity search engines list. You can find the link here.

Each card lists the specific task and links to the suggested tools. First there is a world clock with commonly used time zones for referencing where the threat actor may live based on activity times. There is a cryptocurrency search with likes to address checker and other tools to see if they did bad opsec and linked their personal data to the wallet. SMS services for 2FA when signing up for forums without giving away your real info. Bitcoin tools such as whom they’re registered to. Data & Time tools for world time as well as sun and moon calculators to find out when someone was a certain location. Weather with historical data and world temperatures for finding out where someone is.

WhatsApp tools for SOCMINT to retrieve metadata and generate chats. Copy of identifying combatant affiliatio for identifying state-sponsored threat actors. OSINT Tools/Dashboards contain multiple peoples collects and dashboards plus links to the Framework. This is my favourite section as any other tools you can’t find on this dashboard lives on another. Credit Card / BIN info with lookup databases for finding if they linked, they’re real info. Webcams with links to webcam feeds around the world to observing threat actors in a known location. Email search with disposable emails for sign up as well as several email finder services.

News from Reddit live feed with news which seems to show everything from scripts, opsec, privacy questions etc. Android and iOS Emulator for sandboxing apps for malware analysis. RSS and FEED for collecting data on what a watched group post. Monitoring News/Conflicts/Web for monitoring which hacker groups have aligned themselves with which factions/ governments. Alternative Website Security Search with backup tools for is this website safe, website scanners and other tools. Website Security with commonly used tools like Virustotal, Whois and Malicious URL Scanner.

Determining Age and Race for scanning pictures for age, race and deepfake detection. Discussion Forums for popular osint communities on reddit and beyond. Browser / IP / DNS Leak Test tools for checking that you’ve properly changed them. News with links to news search engines around the world for fact checking. Toolkits / Other Resources with several toolkits listed. Copy of Scientific Search Engines for cross referencing patents and research papers for which types of threat actors would have access to what. Drones & Webcams Databases with links to open webcams and drones for threat actor surveillance.

OSINT Collections is part 2 of listing other people’s collections such as the Digintel Start.me which starts a descent into rabbit hole. Network / Whois / IP / DNS for reverse IP address lookup, port scanners and includes well-known tools like Shodan. Maps & Satellites for observing on the ground footage for finding threat actors also has drone and dash cam footage. Phone Number Search for checking true caller ID and who owns which number if they have it linked to their PII.

Social Media Search with tools for username checkup across platforms. IP Geolocation for finding location of a threat actor from an IP. Technology Lookup / BuiltWith for checking to see what a website was created with or is running. Country with passport index tools for referencing where a threat actor may live. Note Capture Record Screen Shot tools to help with documentation. Clubhouse tools for SOCMINT including room finder. Geo Tools for searching for location meta data. Facebook tools for searching friends, posts and faking data. Twitch tools for checking username and metrics for cross referencing.

Vehicle/ Cars tools for checking plate registration and if the vehicle has been reported stolen. Verifying Online Information tools and best practices reminder. Discord server lists, search functions and bot use. Default Password lists for routers as well as default credentials for services. Translation/Language services that will translate text. Twitter, now called X, tools for checking shadow bans, tweet searches as well as tweet generators for filling a fake profile. Skype tools for checking connected IPs in calls. Materials/ Mac tools for vendor and mac address lookup.

IMINT Tools for checking if an image is edited and editing images. Photo Quality Improvement for upscaling to look for useful information in pictures such as password sticky notes. Hash Decrypt/Encrypt tools for encryption and decryption of passwords. Extra Metadata tools for viewing and extracting metadata from files. Unredactor tool and information on how to reveal redacted information. ID Generator tools for generating fake information for names, SNN, companies and social media profiles. Instagram tools for ID search, data scrapers and location finders. Image Intelligence tools for reserve images searches, camera tracing and removing backgrounds from pictures.

Tiktok tools for searching, exporting comments and downloading videos for analysis. Copy of Stock Photos for referencing against for fake profiles as well as making fake profiles. Reddit tools for searching for posts, users and checking archive. Deep & Dark Web collection of search engines for finding deep and dark web websites. Antivirus and Malware Protection tools for securing systems. Company & Business Info sites to check if a company is legit for just a front for a scam. Breaches/ Leaks tools for checking leaked passwords, emails and PII. Youtube channel crawler, comment finders and metadata. Aviation Movement / Flight Tracker tools to track threat actors' flights so you know when they have changed locations.

Account Killer tools for deleting unwanted data left out in the public. LinkedIn post-date extractors and email searches for cross referencing. People/ Name for searching information on people when you have a full name including public record background checks and who’s related. Browser Extensions for tools like Shodan and Wayback Machine for ease of use. Deepfake / Fake ID tools for creating deepfakes including info and pictures. Video Forensic and Enhancement Tools for cleaning up video so you can properly see anything that can be used. Maritime Movement / Ships tools for ship tracking for following where a threat actor is located.

SMAT (Analyze Hate and Disinformation Online) social media toolkit to find the sources of certain campaigns. File Upload websites where you can upload files to send to others such as malware samples for analyzing. Email Alternatives lists of disposable emails and privacy focused services. Snap Chat tools for SOCMINT for searching names and if one was compromised. Private Search Engine list of privacy focused search engines for searching. Domain Analytics tools to find sub domains and assets listed online. Monitoring Events & Maps tools for monitoring the locations of conflicting including terrorist attacks to know where threat actors may come from.

Face Search Engine tools for facial recognition and finding faces in images. Maps tools for checking maps around the world for cross referencing a threat actors' location. Email Verifier tools to check if an email exists and it's attached to the company it says it’s from. Backlink Checker / Link Analysis tools for checking where a link goes to before you click on it. Website Visual Search tools for searching previous states of a website that may have taken down useful content. HTTP/SSL Certificates Search tools for searching the validity of certificates for checking connection security. Download Websites and Links web scraps, extractors and archives for making a copy of website content.

TryHackMe c4ptur3-th3-fl4g | November 5, 2023

This is a write up for the TryHackMe room called c4ptur3-th3-fl4g which is an Easy room and my personal favourite so far since starting try hack me. At the time of this entry I'm ranked top 3% on the leaderboard. You can find the room here.

Task 1: Translation & Shifting | Tasks can be completed with Cyberchef + ASCII to Text.

Question: c4n y0u c4p7u23 7h3 f149?

Answer: (L33t Sp34k = sub numbers that look like letters) can you capture the flag

Question: 01101100 01100101 01110100 01110011 00100000 01110100 01110010 01111001 00100000 01110011 01101111 01101101 01100101 00100000 01100010 01101001 01101110 01100001 01110010 01111001 00100000 01101111 01110101 01110100 00100001

Answer: (Binary) lets try some binary out!

Question: MJQXGZJTGIQGS4ZAON2XAZLSEBRW63LNN5XCA2LOEBBVIRRHOM======

Answer: (Base 32) base32 is super common in CTF's

Question: RWFjaCBCYXNlNjQgZGlnaXQgcmVwcmVzZW50cyBleGFjdGx5IDYgYml0cyBvZiBkYXRhLg==

Answer: (Base 64) Each Base64 digit represents exactly 6 bits of data.

Question: 68 65 78 61 64 65 63 69 6d 61 6c 20 6f 72 20 62 61 73 65 31 36 3f

Answer: (Hexadecimal / Base 16) hexadecimal or base16?

Question: Ebgngr zr 13 cynprf!

Answer: (Rot 13) Rotate me 13 places!

Question: *@F DA:? >6 C:89E C@F?5 323J C:89E C@F?5 Wcf E:>6DX

Answer: (Rot 47) You spin me right round baby right round (47 times)

Question: - . .-.. . -.-. --- -- -- ..- -. .. -.-. .- - .. --- -. . -. -.-. --- -.. .. -. --.

Answer: (Morse Code) TELECOMMUNICATION ENCODING

Question: 85 110 112 97 99 107 32 116 104 105 115 32 66 67 68

Answer: (Decimal) Unpack this BCD

Question: LS0tLS0gLi0tLS0gLi0tLS0gLS0tLS0gLS0tLS0gLi0tLS0gLi0tLS0gLS0tLS0KLS0tLS0gLi0tLS0gLi0tLS0gLS0tLS0gLS0tLS0gLi0t...

Answer: (Make sure you copy it all or it errors. Base 64 => Morse Code => Binary => Rot47 => ASCII ) Let's make this a bit trickier...
